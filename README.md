# Kişisel Finans ve Harcama Takip Sistemi

Python ve Tkinter kullanılarak geliştirilmiş, kullanıcı dostu bir masaüstü harcama takip uygulamasıdır. Bu proje, günlük harcamaları kategorize ederek kaydetmeyi ve geçmişe dönük yıllık özetler sunmayı amaçlar.

## Özellikler
* **Hızlı Harcama Kaydı:** Tutar, not ve kategori bazlı hızlı veri girişi.
* **Kategori Butonları:** Kullanıcı dostu arayüz ile tek tıkla kategori seçimi.
* **Veri Kalıcılığı:** Harcamalar yerel bir `JSON` dosyasında güvenli bir şekilde saklanır.
* **Aylık Toplam:** Ana ekranda anlık olarak o ayki toplam harcamayı görme.
* **Yıllık Genel Bakış:** Tüm yılı ay ay özetleyen takvim görünümü.
* **Masaüstü Uyumluluğu:** `.exe` formatına dönüştürülebilir ve özel ikon ile kullanılabilir.

## Kullanılan Teknolojiler
* **Dil:** Python 3.x
* **Arayüz (GUI):** Tkinter
* **Veri Formatı:** JSON
* **Kütüphaneler:** `os`, `json`, `datetime`, `messagebox`

Projenin çalışması için bilgisayarınızda Python yüklü olmalıdır.
